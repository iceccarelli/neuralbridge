# Security Policy

## Supported Versions

We are committed to ensuring the security of NeuralBridge. Security patches are backported to the latest major release.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

We take all security bugs in NeuralBridge seriously. Thank you for improving the security of our project. We appreciate your efforts and responsible disclosure and will make every effort to acknowledge your contributions.

To report a security vulnerability, please email us at `security@neuralbridge.ai`.

Please include the following information in your report:

- A description of the vulnerability and its impact.
- Steps to reproduce the vulnerability.
- Any proof-of-concept code.

We will respond to your report within 48 hours and will keep you informed of the progress of our investigation.

**Please do not report security vulnerabilities through public GitHub issues.**
